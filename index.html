I. Information

I.I About IB QRG

The IB Quick Reference Guide is a central location for all important IB information, including exam and paper dates, details about assignments, etc. At the moment, this is not a study guide.

I.II How to use this guide

The DIRECTORY is your fastest way to navigate the site. Click on the section you wish to view. Your browser will automatically scroll to that section seamlessly. To return to the top, click on the RETURN TO TOP link located at the end of a section.

1.2 Criteria

1.2.1 Papers and commentaries

1500 word maximum paper with 1.5 line spacing. Ideally, there should be 1" (2.54 cm) margins around the page. Table of contents page.

Title should be one third into the page. On official copies, your name should not be included.

Page numbers should not appear on the title page, nor the table of contents page.

Use one of the following seven IB concepts.

  • Communication
  • Creativity
  • Perspective
  • Representation
  • Identity
  • Culture
  • Transformation

Bibliography page(s) at the end. To cite your sources, a footnote should be added to where you're citing, and the footnote should cite the source in APA format.

  • If the same exact source (and page) is cited back-to-back: Footnote -> Loc. cit.
  • If the same source, but different page/section: Footnote -> Ibid.
  • If source A is cited once, then source B is cited, then source A is cited: Footnote -> Recite in full.
